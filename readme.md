基于Python3.7编写的一个简易视频播放器

用到的库有：PyQT5   pyAv  (一个基于FFmpeg的底层的库)

安装PyQT：

```
pip3 install PyQt
```

安装pyAv： 

```
pip3 intall pyAv
```

pyAv使用文档:

[1]: https://programtalk.com/vs2/?source=python/8441/PyAV/examples

运行：
直接run mainwid.py
